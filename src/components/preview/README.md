## 架构说明

预览组件目录，实现 UISchema 的实时渲染预览。
支持设备模式切换、主题切换、错误边界处理、未知组件处理等功能。

## 文件索引

| 文件名 | 功能描述 |
|--------|----------|
| index.ts | 组件导出入口 |
| PreviewPanel.tsx | 主预览面板，UISchema 渲染、设备尺寸适配、空状态显示 |
| PreviewPanelWithControls.tsx | 带控制栏的预览面板，集成设备选择器和主题切换 |
| DeviceSelector.tsx | 设备选择器，桌面/平板/手机视图切换 |
| ThemeToggle.tsx | 主题切换组件，浅色/深色/系统主题，持久化存储 |
| ErrorBoundary.tsx | 错误边界组件，捕获渲染错误、优雅降级、未知组件占位符（带建议）、未知组件列表 |
