## 架构说明

布局组件目录，提供三栏可调整宽度的布局系统。支持拖拽调整面板宽度、面板折叠/展开、响应式布局（窄屏标签页模式）。

## 文件索引

| 文件名 | 功能描述 |
|--------|----------|
| ThreeColumnLayout.tsx | 三栏布局主组件，管理 Chat/Editor/Preview 三个面板 |
| ResizablePanel.tsx | 可调整宽度的面板组件，支持拖拽分隔条（水平/垂直方向） |
| PanelHeader.tsx | 面板头部组件，包含标题和折叠/展开按钮 |
| TabBar.tsx | 标签栏组件，用于窄屏模式下的面板切换 |
| index.ts | 模块导出入口 |

## 主要功能

- **可调整宽度**: 拖拽分隔条调整面板宽度百分比
- **可调整高度**: ResizeHandle 支持垂直方向拖拽（用于 Editor 面板内部分割）
- **面板折叠**: 点击折叠按钮隐藏面板，显示折叠指示器
- **响应式布局**: 窄屏（<768px）自动切换为标签页模式
- **状态持久化**: 布局状态自动保存到 localStorage
