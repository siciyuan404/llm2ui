## 架构说明

设置组件目录，包含 LLM 配置相关的设置界面组件。
提供可视化的 LLM 提供商配置、API 密钥管理、模型参数调整、自定义模型管理等功能。

## 文件索引

| 文件名 | 功能描述 |
|--------|----------|
| LLMSettingsDialog.tsx | LLM 设置对话框组件，支持多提供商配置、测试连接、参数调整 |
| ModelManagementDialog.tsx | 模型管理对话框组件，支持自定义模型的添加、编辑、删除和搜索 |
| index.ts | 模块导出入口 |

## 组件说明

### LLMSettingsDialog

LLM 设置对话框，实现以下功能：
- 提供商选择（OpenAI、Anthropic、iFlow、自定义）
- API 密钥输入（密码类型）
- 模型选择下拉菜单（显示预设和自定义模型，带视觉标记区分）
- "管理模型"按钮，打开 ModelManagementDialog
- 温度和最大 token 数配置
- 自定义端点输入
- 系统提示词编辑
- 测试连接功能
- 配置验证和错误显示

### ModelManagementDialog

模型管理对话框，实现以下功能：
- 分组显示预设模型和自定义模型
- 添加自定义模型（模型名称、显示名称、提供商、端点、描述）
- 编辑自定义模型配置
- 删除自定义模型（带确认对话框）
- 搜索/过滤模型列表
- 预设模型显示"预设"标签，自定义模型显示"自定义"标签
- 自定义模型显示编辑/删除按钮
