## 架构说明

配置模块，包含关键词映射配置文件和配置加载器。支持外部化配置和运行时热重载。

## 文件索引

| 文件名 | 功能描述 |
|--------|----------|
| keyword-mappings.json | 关键词到标签/分类的映射配置文件 (Requirements 3.1, 3.4) |
| config-loader.ts | 配置加载器，支持异步加载、验证和热重载 (Requirements 3.2, 3.3, 3.5) |
| config-loader.test.ts | 配置加载器属性测试，验证配置加载健壮性 (Property 3)、配置热重载一致性 (Property 4) |
